# Sticky Notes - Markdown

Simple, native macOS sticky notes with **Markdown** support — colorful notes
that live on your desktop with live preview, code highlighting, global hotkeys,
and optional iCloud sync.

## ⬇︎ Download

**[Download the latest version](../../releases/latest)** → open the `.dmg` and
drag **Sticky Notes - Markdown** into your Applications folder.

The app is signed and notarized by Apple, so it opens with no security warnings,
and it keeps itself up to date automatically after that.

**Requires macOS 26 (Tahoe) or later.**
